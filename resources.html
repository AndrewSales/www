Tools

dtd2xml

Generates an XML representation of an XML 1.0 DTD. Useful for analyzing, normalizing or documenting a DTD.

indenter

An indentation tool for XML documents, written in Python.

schematron4word

Schematron validation for office documents.

schematron-diff

A diffing tool that uses an automatically-generated Schematron schema to produce a delta in SVRL format.
