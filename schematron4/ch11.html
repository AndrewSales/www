Clauses 6.7 (Localization and language fixup)

An implementation shall perform language fixup according to the method defined in section 4.5.6 of XInclude 1.0 when incorporating external definitions (via elements include and extends) and instantiating abstract patterns, abstract rules, diagnostics, and properties.

Clauses 6.8 (Base URI fixup)

An implementation shall perform base URI fixup according to the method defined in section 4.5.5 of XInclude 1.0 when incorporating external definitions (via elements include and extends).
